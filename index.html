Other information

The best solution for the main data is to keep and maintain it on EBT (as it's actually done now) and have possibility to get it in form of data package for 3rd party application.
For other possible regional data, for example, bingo games which are available in several EBTF discussion threads, data file can be attached to the message on the forum.
You can find many regional data files spread on EBTF.
